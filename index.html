NodeKeynote

Esse é um projeto para aprender e aprimorar meus conhecimentos em node.js. A idéia é ter uma ferramenta para fazer apresentações on line, que permitam ter duas visões: apresentador e espectador. Para usá-lo, você precisar ter instalado o node.js (última versão)

$ git clone ...
$ cd your_repo
$ npm install

Projeto

O projeto é bem simples e usa o coffeescript para desenvolver.
